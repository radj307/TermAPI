Replaces all text in the current viewport specified by the given mode with space characters.

Parameters
modeOperation Mode.
  • 0 / CURSOR_TO_END Erases all text from the current cursor position until the end of the viewport.
  • 1 / BEGIN_TO_CURSOR Erases all text from the beginning of the viewport until & including the current cursor position.
  • 2 / ALL_TEXT Erases all text in the entire viewport.
